# 專案介紹
這是一個簡單的支出追蹤 API，旨在幫助用戶有效管理財務狀況。它支持建立、讀取、更新和刪除支出記錄的基本 CRUD 操作。

# 專案目標
- 提供一個直觀易用的支出追蹤系統。
- 支持基本的 CRUD 操作，讓用戶能夠輕鬆管理支出記錄。
- 提供清晰的 API 文檔，便於開發者使用。

# 專案架構
本專案基於 ASP.NET Core Web API 構建，使用 Entity Framework Core 作為 ORM 工具操作資料庫。

- **Controllers**：處理進來的 HTTP 請求並返回適當的響應。
- **Models**：定義資料模型。
- **Repositories**：實現與資料庫的交互邏輯。
- **Migrations**：包含資料庫遷移腳本，用於資料庫結構的版本控制。

# 使用工具
- .NET 8.0 或更高版本
- Visual Studio 2022 或更高版本

# 快速開始
1. 克隆此存儲庫到本地機器。
2. 使用 Visual Studio 打開解決方案文件。
3. 構建解決方案以恢復 NuGet 包。
4. 運行應用程式。
